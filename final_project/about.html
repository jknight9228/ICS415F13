Our webpage uses standard HTML with PHP for interactivity. The login and register function is written in PHP. Logins and passwords are stored in our MySQL database.

In order to grab data from Xbox, we are using Xbox Leaders API
The XboxLeaders API allows developers to access public Xbox LIVE data in order to encourage app development around the Xbox LIVE service. It also is open source and was available for us to use.

XboxLeaders uses JSON to request data from official Xbox servers and returns its back in a format that is readable.
